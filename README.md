# tma_commissioning_sims
Simulation code for commissioning a TMA telescope on-orbit.

Methods used in simulation code include model-based correction, M1 & M2 balancing and Stochastic Parallel Gradient Descent (SPGD).

To run the software you will need a MATLAB R2025b and Ansys Zemax OpticsStudio 2025 R2.04 license.
